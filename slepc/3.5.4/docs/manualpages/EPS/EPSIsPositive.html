EPSIsPositive

Ask if the EPS object corresponds to an eigenvalue problem type that requires a positive (semi-) definite matrix B.

Synopsis

#include "slepceps.h" 
PetscErrorCode EPSIsPositive(EPS eps,PetscBool* is)
Not collective

Input Parameter

eps - the eigenproblem solver context

Output Parameter

is - the answer
