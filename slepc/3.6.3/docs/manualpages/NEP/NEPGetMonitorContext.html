NEPGetMonitorContext

Gets the monitor context, as set by NEPMonitorSet() for the FIRST monitor only.

Synopsis

#include "slepcnep.h" 
PetscErrorCode NEPGetMonitorContext(NEP nep,void **ctx)
Not Collective

Input Parameter

nep - eigensolver context obtained from NEPCreate()

Output Parameter

ctx - monitor context
