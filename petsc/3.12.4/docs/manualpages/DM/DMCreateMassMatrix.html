DMCreateMassMatrix

 Gets mass matrix between two DM objects, M_ij = \int \phi_i \psi_j

Synopsis

#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMCreateMassMatrix(DM dm1, DM dm2, Mat *mat)
Collective on dm1

Input Parameter

dm1 - the DM object
dm2 - the second, finer DM object

Output Parameter

mat -the interpolation
