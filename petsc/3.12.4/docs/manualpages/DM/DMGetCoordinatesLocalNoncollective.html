DMGetCoordinatesLocalNoncollective

 Non-collective version of DMGetCoordinatesLocal(). Fails if global coordinates have been set and DMGetCoordinatesLocalSetUp() not called.

Synopsis

#include "petscdmlabel.h"     
#include "petscds.h"

Not collective

Input Parameter

dm -the DM

Output Parameter

c -coordinate vector
