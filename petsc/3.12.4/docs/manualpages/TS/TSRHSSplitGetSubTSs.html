TSRHSSplitGetSubTSs

 Get an array of all sub-TS contexts.

Synopsis

#include "petscts.h"  
PetscErrorCode TSRHSSplitGetSubTSs(TS ts,PetscInt *n,TS *subts[])
Logically Collective on TS

Output Parameters

n - the number of splits
subksp - the array of TS contexts

Note

After TSRHSSplitGetSubTS() the array of TSs is to be freed by the user with PetscFree() (not the TS just the array that contains them).
