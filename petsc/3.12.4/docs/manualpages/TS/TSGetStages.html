TSGetStages

Synopsis

PetscErrorCode  TSGetStages(TS ts,PetscInt *ns,Vec **Y)

Input Parameter

ts -the TS context obtained from TSCreate()

Output Parameters

ns - the number of stages
Y - the current stage vectors
