ISView

 Displays an index set.

Synopsis

#include "petscis.h" 
PetscErrorCode  ISView(IS is,PetscViewer viewer)
Collective on IS

Input Parameters

is - the index set
viewer - viewer used to display the set, for example PETSC_VIEWER_STDOUT_SELF.

See Also

PetscViewerASCIIOpen()

Level

intermediate
