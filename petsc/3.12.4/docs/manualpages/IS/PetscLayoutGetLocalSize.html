PetscLayoutGetLocalSize

 Gets the local size for a PetscLayout object.

Synopsis

PetscErrorCode  PetscLayoutGetLocalSize(PetscLayout map,PetscInt *n)
Not Collective

Input Parameters

map -pointer to the map

Output Parameters

n -the local size

Notes
