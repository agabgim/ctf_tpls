PCSPAISetCacheSize

 specify cache size in the SPAI preconditioner

Synopsis

#include "petscpc.h" 
PetscErrorCode  PCSPAISetCacheSize(PC pc,int cache_size)

Input Parameters

pc - the preconditioner
n - cache size {0,1,2,3,4,5} (default 5)

Notes

SPAI uses a hash table to cache messages and avoid redundant communication. If suggest always using 5. This parameter is irrelevant in the serial version.

PCSPAI, PCSetType()
