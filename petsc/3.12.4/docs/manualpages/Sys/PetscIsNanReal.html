PetscIsNanReal

 Returns whether the input is a Not-a-Number (NaN) value.

Synopsis

#if defined(PETSC_USE_REAL___FLOAT128)
PetscBool PetscIsNanReal(PetscReal a)

Input Parameter

a -the floating point number

Notes

uses the C99 standard isnan() on systems where it exists. Otherwises uses (a != a), note that some optimizing compiles compile out this form, thus removing the check.

Level

beginner
