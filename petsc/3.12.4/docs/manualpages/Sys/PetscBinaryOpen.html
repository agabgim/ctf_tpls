PetscBinaryOpen

 Opens a PETSc binary file.

Synopsis

PetscErrorCode  PetscBinaryOpen(const char name[],PetscFileMode mode,int *fd)
Not Collective

Input Parameters

name - filename
type - type of binary file, one of FILE_MODE_READ, FILE_MODE_APPEND, FILE_MODE_WRITE

Output Parameter

fd -the file

Notes

Files access with PetscBinaryRead() and PetscBinaryWrite() are ALWAYS written in big-endian format. This means the file can be accessed using PetscBinaryOpen() and PetscBinaryRead() and PetscBinaryWrite() on any machine.
