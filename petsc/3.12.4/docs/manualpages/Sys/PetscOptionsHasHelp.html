PetscOptionsHasHelp

 Determines whether the "-help" option is in the database.

Synopsis

#include "petscsys.h"   
PetscErrorCode PetscOptionsHasHelp(PetscOptions options,PetscBool *set)
Not Collective

Input Parameters

options -options database, use NULL for default global database

Output Parameters
