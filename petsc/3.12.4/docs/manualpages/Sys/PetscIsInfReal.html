PetscIsInfReal

 Returns whether the input is an infinity value.

Synopsis

#if defined(PETSC_USE_REAL___FLOAT128)
PetscBool PetscIsInfReal(PetscReal a)

Input Parameter

a -the floating point number

Notes

uses the C99 standard isinf() on systems where it exists. Otherwises uses (a && a/2 == a), note that some optimizing compiles compile out this form, thus removing the check.

Level

beginner
