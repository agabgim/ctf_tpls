PetscOptionsAllUsed

 Returns a count of the number of options in the database that have never been selected.

Synopsis

#include "petscsys.h"   
PetscErrorCode PetscOptionsAllUsed(PetscOptions options,PetscInt *N)
Not Collective

Input Parameter

options -options database, use NULL for default global database

Output Parameter

N -count of options not used

Notes

The value returned may be different on each process and depends on which options have been processed on the given process

See Also

PetscOptionsView()

Level

advanced
