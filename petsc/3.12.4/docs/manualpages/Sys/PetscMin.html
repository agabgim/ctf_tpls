PetscMin

 Returns minimum of two numbers

Synopsis

#include <petscmath.h>
type PetscMin(type v1,type v2)
Not Collective

Input Parameter

v1 - first value to find minimum of
v2 - second value to find minimum of

Notes

type can be integer or floating point value

See Also

PetscMax(), PetscClipInterval(), PetscAbsInt(), PetscAbsReal(), PetscSqr()

Level

beginner

Location

include/petscmath.h
