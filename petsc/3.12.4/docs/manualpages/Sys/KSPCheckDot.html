KSPCheckDot

 Checks if the result of a dot product used by the corresponding KSP contains Inf or NaN. These indicate that the previous application of the preconditioner generated an error Collective on ksp

Input Parameter

ksp -the linear solver (KSP) context.

Output Parameter

beta -the result of the inner product

Developer Note

this is used to manage returning from KSP solvers whose preconditioners have failed in some way
