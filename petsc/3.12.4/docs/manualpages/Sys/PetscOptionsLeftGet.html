PetscOptionsLeftGet

 Returns all options that were set and never used.

Synopsis

#include "petscsys.h"   
PetscErrorCode PetscOptionsLeftGet(PetscOptions options,PetscInt *N,char **names[],char **values[])
Not Collective

Input Parameter

options -options database, use NULL for default global database

Output Parameter

N - count of options not used
names - names of options not used
values - values of options not used

Notes

Users should call PetscOptionsLeftRestore() to free the memory allocated in this routine Notes: The value returned may be different on each process and depends on which options have been processed on the given process

See Also

PetscOptionsAllUsed(), PetscOptionsLeft()

advanced
