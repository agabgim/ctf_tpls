PetscImaginaryPart

 Returns the imaginary part of a PetscScalar

Synopsis

#include <petscmath.h>
PetscReal PetscImaginaryPart(PetscScalar v)
Not Collective

Input Parameter

v -value to find the imaginary part of

Notes

If PETSc was configured for real numbers then this always returns the value 0
