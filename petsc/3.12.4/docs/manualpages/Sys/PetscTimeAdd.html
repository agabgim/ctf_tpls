PetscTimeAdd

 Adds the current time of day (in seconds) to the value v.

Synopsis

#include <petsctime.h>
PetscErrorCode PetscTimeAdd(PetscLogDouble *v)
Not Collective

Input Parameter

v -time counter

Output Parameter

v -time counter (v = v + current time)

Notes

Since the PETSc libraries incorporate timing of phases and operations, we do not ever recommend using PetscTimeAdd(). The options database command -log_view activate PETSc library timing. See Users-Manual: Chapter 13 Profiling for more details.
