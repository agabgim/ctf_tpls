MatKAIJSetS

 Set the S matrix describing the shift action of the KAIJ matrix

Synopsis

#include "petscmat.h" 
PetscErrorCode MatKAIJSetS(Mat A,PetscInt p,PetscInt q,const PetscScalar S[])
Logically Collective; the entire S is stored independently on all processes.

Input Parameters

A - the KAIJ matrix
p - the number of rows in S
q - the number of columns in S
S - the S matrix, in form of a scalar array in column-major format

Notes: The dimensions p and q must match those of the transformation matrix T associated with the KAIJ matrix. The S matrix is copied, so the user can destroy this array.

See Also

MatKAIJGetS(), MatKAIJSetT(), MatKAIJSetAIJ()
