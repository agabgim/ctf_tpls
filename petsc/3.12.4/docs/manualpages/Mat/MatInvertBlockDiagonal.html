MatInvertBlockDiagonal

 Inverts the block diagonal entries.

Synopsis

#include "petscmat.h" 
PetscErrorCode MatInvertBlockDiagonal(Mat mat,const PetscScalar **values)
Collective on Mat

Input Parameters

mat -the matrix

Output Parameters

values -the block inverses in column major order (FORTRAN-like)
