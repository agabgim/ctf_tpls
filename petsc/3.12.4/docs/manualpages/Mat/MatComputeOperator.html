MatComputeOperator

 Computes the explicit matrix

Synopsis

#include "petscmat.h"  
PetscErrorCode  MatComputeOperator(Mat inmat,MatType mattype,Mat *mat)
Collective on Mat

Input Parameter

inmat - the matrix
mattype - the matrix type for the explicit operator

Output Parameter

mat -the explict operator

Notes

This computation is done by applying the operators to columns of the identity matrix. This routine is costly in general, and is recommended for use only with relatively small systems. Currently, this routine uses a dense matrix format if mattype == NULL.

advanced
