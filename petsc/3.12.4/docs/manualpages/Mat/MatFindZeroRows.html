MatFindZeroRows

 Locate all rows that are completely zero in the matrix

Synopsis

#include "petscmat.h" 
PetscErrorCode MatFindZeroRows(Mat mat,IS *zerorows)

Input Parameter

A -the matrix

Output Parameter

zerorows -the rows that are completely zero

Notes

zerorows is set to NULL if no rows are zero.
