MatRestoreRowUpperTriangular

 Disable calls to MatGetRow() for matrix in MATSBAIJ format.

Synopsis

#include "petscmat.h" 
PetscErrorCode MatRestoreRowUpperTriangular(Mat mat)
Not Collective

Input Parameters

Notes
