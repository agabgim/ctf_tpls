MatMissingDiagonal

 Determine if sparse matrix is missing a diagonal entry (or block entry for BAIJ matrices)

Synopsis

PetscErrorCode MatMissingDiagonal(Mat mat,PetscBool *missing,PetscInt *dd)
Not Collective

Input Parameter

mat -the matrix

Output Parameters
