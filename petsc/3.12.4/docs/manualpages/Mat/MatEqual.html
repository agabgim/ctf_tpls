MatEqual

 Compares two matrices.

Synopsis

PetscErrorCode MatEqual(Mat A,Mat B,PetscBool  *flg)
Collective on Mat

Input Parameters

A - the first matrix
B - the second matrix

Output Parameter

flg -PETSC_TRUE if the matrices are equal; PETSC_FALSE otherwise.
