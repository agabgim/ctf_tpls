VecRestoreArray

 Restores a vector after VecGetArray() has been called.

Synopsis

#include "petscvec.h"   
PetscErrorCode VecRestoreArray(Vec x,PetscScalar **a)
Logically Collective on Vec

Input Parameters

x - the vector
a - location of pointer to array obtained from VecGetArray()
