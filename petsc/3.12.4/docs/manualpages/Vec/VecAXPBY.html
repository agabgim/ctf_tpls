VecAXPBY

 Computes y = alpha x + beta y.

Synopsis

#include "petscvec.h"   
PetscErrorCode  VecAXPBY(Vec y,PetscScalar alpha,PetscScalar beta,Vec x)
Logically Collective on Vec

Input Parameters

alpha,beta - the scalars
x, y - the vectors

Output Parameter

y -output vector

Notes

x and y MUST be different vectors The implementation is optimized for alpha and/or beta values of 0.0 and 1.0

See Also
