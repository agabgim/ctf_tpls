VecAYPX

 Computes y = x + beta y.

Synopsis

#include "petscvec.h"   
PetscErrorCode  VecAYPX(Vec y,PetscScalar beta,Vec x)
Logically Collective on Vec

Input Parameters

beta - the scalar
x, y - the vectors

Output Parameter

y -output vector

Notes

x and y MUST be different vectors The implementation is optimized for beta of -1.0, 0.0, and 1.0

See Also

VecMAXPY(), VecWAXPY(), VecAXPY(), VecAXPBYPCZ(), VecAXPBY()

intermediate
