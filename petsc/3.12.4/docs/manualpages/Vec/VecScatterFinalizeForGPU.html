VecScatterFinalizeForGPU

 Finalizes a generalized scatter from one vector to another for GPU based computation.

Synopsis

#include "petscvec.h"    
PETSC_EXTERN PetscErrorCode VecScatterFinalizeForGPU(VecScatter inctx)

Input Parameter

inctx -scatter context generated by VecScatterCreate()

Notes

Effectively, this function resets the temporary buffer flags. Currently, this only used in the context of the parallel SpMV call in in MatMult_MPIAIJCUSPARSE Once the MatMultAdd is finished, the GPU temporary buffers used for messaging are no longer valid.
