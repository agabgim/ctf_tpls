VecRestoreArrayWrite

Synopsis

PetscErrorCode VecRestoreArrayWrite(Vec x,PetscScalar **a)
Logically Collective on Vec

Input Parameters

x - the vector
a - location of pointer to array obtained from VecGetArray()
