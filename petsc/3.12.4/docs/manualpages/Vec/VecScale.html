VecScale

 Scales a vector.

Synopsis

#include "petscvec.h"   
PetscErrorCode  VecScale(Vec x, PetscScalar alpha)
Not collective on Vec

Input Parameters

x - the vector
alpha - the scalar

Output Parameter

x -the scaled vector

Note

For a vector with n components, VecScale() computes 
     x[i] = alpha * x[i], for i=1,...,n.

Level

intermediate
