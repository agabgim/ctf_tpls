DMPlexSetMigrationSF

 Sets the SF for migrating from a parent DM into this DM

Synopsis

#include "petscdmplex.h"   
PetscErrorCode DMPlexSetMigrationSF(DM dm, PetscSF migrationSF)

Input Parameters

dm - The DM
naturalSF - The PetscSF

Note: It is necessary to call this in order to have DMCreateSubDM() or DMCreateSuperDM() build the Global-To-Natural map

See Also

DMPlexDistribute(), DMPlexDistributeField(), DMPlexCreateMigrationSF(), DMPlexGetMigrationSF()
