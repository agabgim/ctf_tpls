DMPlexSetScale

 Set the scale for the specified fundamental unit

Synopsis

#include "petscdmplex.h"   
PetscErrorCode DMPlexSetScale(DM dm, PetscUnit unit, PetscReal scale)
Not collective

Input Arguments

dm - the DM
unit - The SI unit
scale - The value used to scale all quantities with this unit
