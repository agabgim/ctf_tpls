DMPlexCheckPointSF

 Check that several necessary conditions are met for the point SF of this plex.

Synopsis

#include "petscdmplex.h"   
PetscErrorCode DMPlexCheckPointSF(DM dm)

Input Parameters

dm -The DMPlex object

Notes

This is mainly intended for debugging/testing purposes. It currently checks only meshes with no partition overlapping.

See Also
