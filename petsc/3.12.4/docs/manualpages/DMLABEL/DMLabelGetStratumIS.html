DMLabelGetStratumIS

 Get an IS with the stratum points

Synopsis

#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelGetStratumIS(DMLabel label, PetscInt value, IS *points)
Not collective

Input Parameters

label - the DMLabel
value - the stratum value

Output Paramater

points -The stratum points

Notes

The output IS should be destroyed when no longer needed. Returns NULL if the stratum is empty.
