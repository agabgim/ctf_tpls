DMLabelCreate

 Create a DMLabel object, which is a multimap

Synopsis

#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelCreate(MPI_Comm comm, const char name[], DMLabel *label)
Collective

Input parameters

comm - The communicator, usually PETSC_COMM_SELF
name - The label name

Output parameter

label -The DMLabel

Notes

The label name is actually usual PetscObject name. One can get/set it with PetscObjectGetName()/PetscObjectSetName().
