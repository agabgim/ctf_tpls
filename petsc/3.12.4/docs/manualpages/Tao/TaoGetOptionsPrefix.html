TaoGetOptionsPrefix

 Gets the prefix used for searching for all TAO options in the database

Synopsis

#include "petsctao.h" 
PetscErrorCode TaoGetOptionsPrefix(Tao tao, const char *p[])
Not Collective

Input Parameters

tao -the Tao context

Output Parameters

prefix -pointer to the prefix string used is returned

Notes

On the fortran side, the user should pass in a string 'prefix' of sufficient length to hold the prefix.

See Also

TaoSetOptionsPrefix(), TaoAppendOptionsPrefix()
