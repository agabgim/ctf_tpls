TaoComputeResidual

 Computes a least-squares residual vector at a given point

Synopsis

#include "petsctao.h" 
PetscErrorCode TaoComputeResidual(Tao tao, Vec X, Vec F)
Collective on Tao

Input Parameters

tao - the Tao context
X - input vector

Output Parameter

f -Objective vector at X

Notes

TaoComputeResidual() is typically used within minimization implementations, so most users would not generally call this routine themselves.
