SNESNGSSetSweeps

 Sets the number of sweeps of GS to use.

Synopsis

#include "petscsnes.h"  
PetscErrorCode SNESNGSSetSweeps(SNES snes, PetscInt sweeps)

Input Parameters

snes - the SNES context
sweeps - the number of sweeps of GS to perform.
