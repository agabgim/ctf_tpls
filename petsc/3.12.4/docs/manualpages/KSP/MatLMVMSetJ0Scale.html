MatLMVMSetJ0Scale

 Allows the user to define a scalar value mu such that J0 = mu*I.

Synopsis

#include "petscksp.h" 
PetscErrorCode MatLMVMSetJ0Scale(Mat B, PetscReal scale)

Input Parameters

B - An LMVM-type matrix
scale - Scalar value mu that defines the initial Jacobian
