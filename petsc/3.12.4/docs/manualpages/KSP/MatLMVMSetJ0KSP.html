MatLMVMSetJ0KSP

 Allows the user to provide a pre-configured KSP solver for the initial inverse-Jacobian approximation. This KSP solver should already contain all the operators necessary to perform the inversion. The LMVM matrix only calls KSPSolve() without changing any other options.

Synopsis

#include "petscksp.h" 
PetscErrorCode MatLMVMSetJ0KSP(Mat B, KSP J0ksp)

Input Parameters

B - An LMVM-type matrix
J0ksp - KSP solver for the initial inverse-Jacobian application
