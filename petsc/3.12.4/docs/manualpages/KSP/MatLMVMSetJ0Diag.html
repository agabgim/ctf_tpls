MatLMVMSetJ0Diag

 Allows the user to define a vector V such that J0 = diag(V).

Synopsis

#include "petscksp.h" 
PetscErrorCode MatLMVMSetJ0Diag(Mat B, Vec V)

Input Parameters

B - An LMVM-type matrix
V - Vector that defines the diagonal of the initial Jacobian
